# v0.3.1 Workspace Release Correction

Date: 2026-06-10
Milestone: `0.3.0`
Issue: #70

## Summary

`v0.3.1` is the corrective release for the `0.3.0` milestone. The previous
`v0.3.0` tag and GitHub Release were created, but only
`bluetape-rs-codec@0.3.0` was published to crates.io. The rest of the workspace
was still versioned as `0.1.1` or `0.2.0`, so the release did not match the
milestone-level package intent.

This release aligns every current workspace package to `0.3.1` and publishes
the whole workspace as the `0.3.x` stable line.

## Release Notes

- `v0.3.0` is immutable because `bluetape-rs-codec@0.3.0` has already been
  published to crates.io.
- `v0.3.1` supersedes `v0.3.0` as the complete workspace release for the
  `0.3.0` milestone.
- The root facade and all focused crates use the same package version.
